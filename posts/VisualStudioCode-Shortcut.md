### VSCode 단축키

- 에디터 수정 관련

1. 터미널 콘솔 화면 on/Off => Ctrl + "J"

2. 파일 탐색기 on/Off => Ctrl + "B"

- 코드 수정 관련

1. 행 주석 처리 => Ctrl + "/"

2. 묶음 주석 처리 => Alt + Shift + "A"

3. 포인트 멀티 선택 => ALT + Click

4. 행 위치 변경 => ALT + up/ Down 키

5. 이전/이후 선택 => ALT + left/ Right 키
